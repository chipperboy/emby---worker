# 更新日志（CHANGELOG）

本文档记录项目的重要变更。

## [2026-03] - 首次公开整理版本

### 新增
- 节点显示名称 `displayName`（支持中文）
### 变更
- 部署模式简化为**全反代单模式**
- 后台编辑弹窗移除“统一/分离”模式选择
- 卡片长标题超出时改为省略显示（避免撑出布局）
- 3xx 跳转改写增强，兼容中文/空格 URL
- 请求头转发策略优化，提升第三方播放器兼容性

### 修复
- 修复编辑节点后排序丢失（保留 `rank`）
- 修复节点改名后收藏状态丢失（保留 `fav`）
- 修复筛选函数返回值问题导致的列表异常
- 修复多处 3xx 改写与 `__raw__` 转发边界问题

### 说明
- 某些上游签名直链可能因来源 IP / 签名策略导致全反代 `403`。
- 如遇此类上游，需要上游策略放行，或采用回落方案。

## [未发布]
- 持续优化第三方播放器兼容与特殊上游适配。
